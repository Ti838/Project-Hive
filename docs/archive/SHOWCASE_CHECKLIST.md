## 🚨 EMERGENCY FIXES

### If something breaks during demo:
1. **Stay calm** - "Let me refresh that"
2. **Have backup** - Other pages work
3. **Explain** - "This is a known edge case we're addressing"
4. **Move on** - Don't dwell on issues

### Common Issues & Quick Fixes:
| Issue | Fix |
|-------|-----|
| Page won't load | Hard refresh (Ctrl+F5) |
| Mobile view broken | Check width is < 768px |
| CSS not applied | Clear cache, reload |
| JavaScript error | Check console, reload page |
